# 🍽️ Meu Cantinho de Receitas

Um site de receitas desenvolvido com HTML como parte dos meus estudos de desenvolvimento web.

## 📖 Sobre o projeto

O **Meu cantinho de Receitas** é um site criado para compartilhar receitas deliciosas e fáceis de preparar. O projeto foi desenvolvido utilizando apenas HTML, com foco na prática de estruturação de páginas web e utilização de tags semânticas.

Os usuários podem navegar pela página inicial, explorar categorias de receitas e acessar páginas individuais com o passo a passo completo de cada prato.

## ✨ Funcionalidades

- Página inicial com apresentação do site.
- Navegação entre páginas através de links.
- Categoria de receitas diversas.
- Páginas individuais para cada receita.
- Estrutura organizada utilizando HTML semântico.

## 🛠️ Tecnologias utilizadas

- HTML5
